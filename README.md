Analyze your bundle size and get AI-powered suggestions to reduce it.

## Install

```bash
npm install -g ai-bundle-check
```

## Usage

```bash
npx ai-bundle-check
npx ai-bundle-check ./my-project/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
